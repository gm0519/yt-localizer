# ✍️ LAFIENA SONGWRITING GUIDE

---

# 📌 기본 구성 포맷

([Hook])
([unique (악기명) only intro])
[verse 1]
([pre-chorus 1])
[chorus 1 (+hook)]
[verse 2]
([pre-chorus 2])
[chorus 2 (+hook)]
[bridge]
[chorus 3 (+hook)]
([outro] (+hook))

※ () = 선택사항 / 모든 곡 동일 구조 금지 — 파트 순서·길이·반복 횟수 곡마다 다르게
※ 맨 앞의 [Hook] 또는 [unique (악기명) intro] 둘 중의 1개 선택 필수

---

# 📏 가사 규칙

- 영어만 / 100% 오리지널
- 100~130 words 목표 / 최대 130 words (엄격한 상한)
- 기존 곡의 문장 구조·리듬·단어 패턴과 절대 겹치지 않게
- 저작권 걸리지 않게 절대로! 기존 유명한 곡 그대로 사용 절대 금지!!!

---

# ✏️ 가사 기호 활용 규칙

기호를 적절히 사용해 곡의 감정·리듬·뉘앙스를 시각화한다.

/ — 같은 박자 안에서 빠르게 이어지는 흐름 또는 선택적 감정
예: "stay / go / I don't know anymore"
예: "you laughed / I froze / the room forgot to breathe"

~ — 여운 / 길게 늘이는 음 / 감정이 흐르듯 이어지는 느낌
예: "and I still think about you~"
예: "somewhere close to something real~"

() — 속마음 / 혼잣말 / 백보컬 레이어 / 보조 멜로디 라인
예: "(why do I keep coming back)"
예: "I said I'm fine (I'm really not)"
예: chorus 뒤 백보컬: "(still here, still here)"

사용 원칙:

- 한 섹션에 1~2개 이내 — 과하면 오히려 노이즈
- 감정 고조 순간 또는 여백이 필요한 순간에만
- [ad-lib] 태그와 () 병행 사용 가능

---

# 🌬️ 여백 설계 원칙

- 한 줄의 음절 수를 의도적으로 줄인다
- 코러스는 짧고 강하게 — 짧은 훅 3회 반복이 긴 코러스보다 효과적
- 싱랩 구간 직후 반드시 짧은 훅 또는 여백 배치

여백 태그:

- [instrumental break] — 악기 채우는 구간 (2~4bar)
- [instrumental 2bar] — 짧은 여백 (싱랩 직후)
- [ad-lib] — 보컬 즉흥 감정 표현

---

# ⏱️ 길이 보장

목표: 2:50~3:50 / 최소: 2:30
단어 수를 줄이는 대신 구조와 반복으로 길이 확보

필수 구조 요소:

- Verse 1 (4~6줄) + Verse 2 (4~6줄)
- Pre-chorus 또는 verse extension (2~4줄)
- Chorus 최소 3회 반복
- Bridge (3~5줄)
- [instrumental break] 또는 [ad-lib] 여백 구간
- Final chorus + outro

❌ Chorus 2회 반복 금지
❌ Bridge 누락 금지
❌ 가사로만 시간 채우기 금지

---

# 🔄 구조 변주 규칙 (하나의 플레이리스트 내부)

변주 가능 요소:

- Pre-chorus 위치 (앞 / 뒤 / 생략)
- Hook 위치 (verse 전 / chorus 후 / 독립)
- Rap / Melodic rap 섹션 (선택적 사용)
- Verse extension (1-1 / 2-1 split vs single block)
- Outro 무게 (light fade / full outro / hook repeat)

---

# 🎵 작사 스타일 원칙

- 어조: 서정적이고 일상적 — talking / texting / thinking out loud
- 운율: 음절 수 · 리듬감 · 운율 3박자 조화
- Flow: syncopated / triplet / conversational / multisyllabic internal rhymes

---

# 📝 출력 형식

- 트랙 제목: 첫 글자만 대문자
- 태그 표기: [verse 1] / [chorus 2] / [bridge] / [outro]

---

# 🎤 TRACK EXECUTION RULES

## Hook / Intro 시작 규칙

**Hook 시작:**

- 가사 맨 앞에 아래 태그 1개만 출력 후 바로 가사 시작
- `[hook-vocal starts clean, no humming, no pre-vocalization, no breath, no ad-libs, no ornament, no melisma, dry vocal entry on first word]`
- ❌ [hook] 태그 별도 추가 금지 / 훅 태그 중복 금지

**Intro 시작 (05·07·11번 트랙 고정):**

- 가사 맨 앞에 아래 태그 출력
- `[unique [악기명] only intro — 2 bars, solo instrumental only, no vocals, no humming, no full band, no ornament, no electronic fill, clean entry]`
- 악기 예시: acoustic guitar / upright bass / trumpet / muted trumpet / flugelhorn / harmonica / bass

❌ Hook + Intro 동시 사용 절대 금지

## 듀엣 가사 규칙

- 번갈아 파트 + 동시 화음 파트 반드시 포함
- 보컬 구분은 반드시 구조태그 안에만 표기
  → `[verse 1 - male]` / `[chorus 1 - together]`
- ❌ 구조태그 밖에 `- Male` / `- Together` 단독 표기 절대 금지

## 플리 전체 구조 다양성 규칙

- 동일 가사 구조 2곡 연속 금지
- pre-chorus → chorus 패턴 13곡 전체 반복 금지
- bridge 위치 고정 금지 (chorus 2 직후 고정 패턴 금지)
- verse 확장(1-1 / 2-1) / ad-lib / instrumental break 삽입 위치 분산
- 13곡 중 최소 5가지 다른 구조 패턴 사용
