# 📌 !플리

## STEP 1 — 타이틀 · 날짜

**타이틀:**

- 있으면 → 그대로 사용, 타이틀과 상관 없더라도, 트랙 가사는 항상 풋풋한 사랑 스토리로 작성
- 없고 주제 있으면 → 주제 기반 자동 생성
- 둘 다 없으면 → 풋풋한 사랑 스토리 랜덤 선택 후 자동 생성

**날짜:** 반드시 질문. 미입력 시 오늘 날짜 자동 적용 (YYMMDD)

## STEP 2 — 카테고리 선택

ask_user_input으로 제시:
① 그루브 ② 힙합 ③ 이지팝1 ④ 이지팝2 ⑤ 시니어팝
미선택 시 → ① 그루브 자동 적용
힙합·시니어팝 현재 비활성

⚠️ 가사·스타일 프롬프트는 세트 요청 전까지 절대 생성 금지

## 출력 1 — 플리 설명글

- 7~9줄 / 한 문장 15자 이내 권장 / 단문 / 줄바꿈
- 한글 / 나열·해시태그 금지
- 톤: 친구가 플리 링크 보내며 툭 던지는 말투
- ~요/~예요/~돼요 허용 / ~습니다·~합니다 금지
- 명사컷 병행 가능
- 구조: ①감각적 훅 한 줄 → ②음악 느낌 묘사 → ③어울리는 상황 대구 → ④가볍게 마무리
- 이모지 자연스럽게 / 색상 다양하게
- 마무리 감사·또 와주세요 금지
- 코드블럭으로 출력

❌ 금지: "세련된 노동요" 강삽입 / 노골적 유도 문장 / "온도" 류 감성 단어 / SEO 직접 유도

SEO (자연스러운 문장으로만 / 나열·해시태그 금지 / 동일 키워드 2회 이상 반복 금지):
노동요 · 일할 때 듣는 음악 · 집중력 높이는 음악 · 공부할 때 듣기 좋은 음악 · 팝 플레이리스트 · 내적댄스 · 둠칫 · 드라이브 음악 · 기분 좋아지는 음악 · 영어 팝송 추천 · 카페 음악

## 출력 2 — 트랙리스트 (13곡)

형식: 번호 / 제목 / 보컬 / BPM
제목 규칙: lafiena-master.md → SONG TITLE RULE 참조

**보컬 포지션 (고정 — 절대 변경 금지):**

- Male: 01·02·05·06·07·09·10·13
- Female: 04·08·11
- Duet: 03·12
- Duet: 번갈아 파트 + 동시 화음 파트 필수

**Hook / Intro 지정:**

- Hook 시작: Intro 지정 트랙 제외 전부 (10곡)
- Intro 시작: 05·07·11 고정

**TITLE CONFLICT CHECK (4단계 필수):**

1. 13곡 내부 중복 확인
2. lafiena-track-title-history.md view + conversation_search로 과거 전체 확인
   → 동일·1단어 차이 이내 교체 / 동일 단어 반복 사용 금지
3. 각 타이틀 + "song" 웹서치로 유명곡 충돌 확인
4. 통과한 타이틀만 확정

## 마무리

→ ask_user_input: [!세트1 시작]
